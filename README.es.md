# Dracula Variants para [Visual Studio Code](https://code.visualstudio.com)

🌍 [English](./README.md) | [中文](./README.zh-CN.md) | [日本語](./README.ja.md) | [한국어](./README.ko.md)

> Temas oscuro y claro para [Visual Studio Code](https://code.visualstudio.com), basados en [Dracula](https://draculatheme.com).

## Temas

Esta extensión incluye **tres** variantes de tema:

| Tema | Tipo | Descripción |
|------|------|-------------|
| **Dracula Theme** | Oscuro | El clásico tema oscuro de Dracula |
| **Dracula Theme Soft** | Oscuro | Una variante desaturada para reducir la fatiga visual |
| **Dracula Theme Light** | Claro | Una variante clara que conserva la paleta de colores característica de Dracula |

### Dracula Light

La variante clara invierte el fondo y el primer plano, mientras profundiza todos los colores de sintaxis para mantener el contraste WCAG AA sobre un fondo claro. El ADN cromático de Dracula — púrpura, rosa, cian, verde — se conserva por completo.

**Paleta de colores:**

## Instalación

### Desde el código fuente

1. Clona este repositorio
2. Copia la carpeta en el directorio de extensiones de VS Code
3. Reinicia VS Code
4. Abre **Preferencias: Tema de color** y selecciona la variante que prefieras

### Desarrollo

```bash
npm install
npm run build
```

Presiona `F5` en VS Code para iniciar el Extension Development Host y previsualizar los temas.
