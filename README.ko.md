# Dracula Variants — [Visual Studio Code](https://code.visualstudio.com) 테마

🌍 [English](./README.md) | [中文](./README.zh-CN.md) | [Español](./README.es.md) | [日本語](./README.ja.md)

> [Dracula](https://draculatheme.com) 기반의 [Visual Studio Code](https://code.visualstudio.com) 다크 & 라이트 테마.

## 테마

이 확장 프로그램에는 **세 가지** 테마 변형이 포함되어 있습니다:

| 테마 | 유형 | 설명 |
|------|------|------|
| **Dracula Theme** | 다크 | 클래식 Dracula 다크 테마 |
| **Dracula Theme Soft** | 다크 | 눈의 피로를 줄이기 위한 저채도 변형 |
| **Dracula Theme Light** | 라이트 | Dracula의 시그니처 컬러 팔레트를 유지한 라이트 변형 |

### Dracula Light

라이트 변형은 배경색과 전경색을 반전시키고, 모든 구문 색상을 깊게 조정하여 밝은 배경에서도 WCAG AA 대비 기준을 충족합니다. Dracula의 컬러 DNA — 퍼플, 핑크, 시안, 그린 — 이 완전히 보존됩니다.

**컬러 팔레트:**

| 역할 | 다크 | 라이트 |
|------|------|--------|
| 배경 | `#282A36` | `#F8F8F2` |
| 전경 | `#F8F8F2` | `#282A36` |
| 키워드 | `#FF79C6` | `#C0358B` |
| 함수 | `#50FA7B` | `#1B8A3A` |
| 문자열 | `#F1FA8C` | `#9B7E00` |
| 상수 | `#BD93F9` | `#7C3AED` |
| 타입 | `#8BE9FD` | `#0C7E8F` |
| 주석 | `#6272A4` | `#7E8CAD` |

## 설치

### 소스에서 설치

1. 이 리포지토리를 클론
2. 폴더를 VS Code 확장 프로그램 디렉토리에 복사
3. VS Code를 재시작
4. **기본 설정: 색 테마** 를 열고 원하는 변형을 선택

### 개발

```bash
npm install
npm run build
```

VS Code에서 `F5`를 눌러 Extension Development Host를 실행하고 테마를 미리 볼 수 있습니다.
