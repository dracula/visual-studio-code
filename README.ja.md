# Dracula Variants — [Visual Studio Code](https://code.visualstudio.com) テーマ

🌍 [English](./README.md) | [中文](./README.zh-CN.md) | [Español](./README.es.md) | [한국어](./README.ko.md)

> [Dracula](https://draculatheme.com) をベースにした [Visual Studio Code](https://code.visualstudio.com) 用ダーク＆ライトテーマ。

## テーマ

この拡張機能には **3つ** のテーマバリエーションが含まれています：

| テーマ | タイプ | 説明 |
|--------|--------|------|
| **Dracula Theme** | ダーク | クラシックな Dracula ダークテーマ |
| **Dracula Theme Soft** | ダーク | 目の疲れを軽減する低彩度バリアント |
| **Dracula Theme Light** | ライト | Dracula の象徴的なカラーパレットを維持したライトバリアント |

### Dracula Light

ライトバリアントは背景色と前景色を反転させ、すべてのシンタックスカラーを深みのある色調に調整することで、明るい背景上でも WCAG AA コントラスト基準を満たしています。Dracula のカラー DNA — パープル、ピンク、シアン、グリーン — は完全に保持されています。

**カラーパレット：**

| 役割 | ダーク | ライト |
|------|--------|--------|
| 背景 | `#282A36` | `#F8F8F2` |
| 前景 | `#F8F8F2` | `#282A36` |
| キーワード | `#FF79C6` | `#C0358B` |

## インストール

### ソースからインストール

1. このリポジトリをクローン
2. フォルダを VS Code の拡張機能ディレクトリにコピー
3. VS Code を再起動
4. **基本設定：配色テーマ** を開き、お好みのバリアントを選択

### 開発

```bash
npm install
npm run build
```

VS Code で `F5` を押すと Extension Development Host が起動し、テーマをプレビューできます。
